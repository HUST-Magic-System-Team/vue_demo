<pre>

**简要描述：** 

1.用户登陆接口

**请求URL：** 
- ` http://100.100.100.100:8080/api/login/login `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|phone |是  |string |手机号   |
|smscode |是  |string | 短信验证码    |


 **返回示例**

```
  {
    "error": 0,
	"errmsg":'',

		"id": "1",			 #用户id
```
